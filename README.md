# Agent skills for consumers

AI should benefit consumers, not just businesses.

This repository provides skills to help consumers make good choices about whom
they choose to do business with and how.

| Skill | Description |
| --- | --- |
| drop-shipper-analysis | Evaluate whether an online store is a dropshipper or dropshipping scam. |
| read-privacy-policy | Summarize a privacy policy and highlight consumer-unfriendly behavior. |
